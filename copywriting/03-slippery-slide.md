# Slippery Slide - Structure narrative qui propulse vers l'action

**Tactique** : Chaque phrase doit donner envie de lire la suivante jusqu'au CTA

**Principe** : Créer un momentum narratif où le lecteur ne peut pas s'arrêter

---

## Structure appliquée à la page d'accueil

### 1. HEADLINE PRINCIPAL (promesse)
```
Chronique des Premiers Hommes
Tome 1 : L'Éveil des Âmes
```
→ Établit le cadre narratif

### 2. HOOK IMMÉDIAT (tension qui accroche)
```
Le loup vous fixe. Vous devez prendre sa vie pour survivre.
Mais dans ses yeux, vous voyez... un ami.
```
→ **Slippery slide** : Chaque phrase crée curiosité pour la suivante
- "Le loup vous fixe" → attire l'attention
- "Vous devez prendre sa vie" → tension monte
- "Mais..." → retournement inattendu
- "un ami" → pic émotionnel

### 3. INVITATION (Bienvenue)
```
Bienvenue dans Chronique des Premiers Hommes.
```
→ Transition douce vers les CTAs

### 4. CTA PRIMAIRE (action avec bénéfice clair)
```
[Lire le prologue gratuitement]
```
→ Bénéfice immédiat + sans risque

### 5. CTA SECONDAIRE (alternative d'achat)
```
[Commander sur Amazon]
```
→ Pour les déjà convaincus

## Below the Fold - Continuation du slide

### 6. RÉINTRODUCTION (setup + tension)
```
De la forêt à perte de vue, des clans vivant en harmonie avec la
nature sauvage... et pourtant, un équilibre millénaire sur le point
de se briser.
```
→ **Slippery slide** :
- "forêt à perte de vue" → image immersive
- "harmonie" → setup positif
- "et pourtant..." → tease le conflit
- "sur le point de se briser" → urgence narrative

### 7. SOCIAL PROOF (validation externe)
```
[3 testimonials des lecteurs]
```
→ Preuve que l'expérience promise est réelle

### 8. LIEN AMAZON (crédibilité externe)
```
Voir tous les avis sur Amazon →
```
→ Redirection vers validation externe (10 reviews 4.8/5)

## Principes du Slippery Slide appliqués

### ✅ Ce qui fonctionne

1. **Tension immédiate** : Le loup + le chasseur = conflit instantané
2. **Retournement** : "Mais... un ami" = surprise émotionnelle
3. **Ellipse narrative** : "..." = lecteur complète mentalement
4. **Contraste positif/négatif** : "harmonie" puis "se briser"
5. **Pas de friction** : Chaque élément mène naturellement au suivant

### ❌ Erreurs évitées

1. ~~"Une saga épique de fantasy"~~ → Trop générique, pas de slide
2. ~~"10 lecteurs l'adorent"~~ → Interruption du flow narratif
3. ~~"Découvrir l'univers"~~ → CTA vague sans bénéfice clair

---

## Techniques de "slide" utilisées

| Technique | Où | Comment |
|-----------|-----|---------|
| **Question implicite** | "Le loup vous fixe" | Lecteur se demande "et après ?" |
| **Retournement** | "Mais... un ami" | Subvertit l'attente, crée surprise |
| **Suspension** | "..." | Force le cerveau à vouloir la suite |
| **Contraste** | "harmonie" → "se briser" | Tension narrative |
| **Tease** | "et pourtant..." | Annonce un problème à découvrir |

---

## Mesure du succès

**Objectif** : Le lecteur doit arriver au CTA sans réfléchir consciemment

**Test mental** :
- Peut-on s'arrêter après "Le loup vous fixe" ? → NON
- Peut-on s'arrêter après "Vous devez prendre sa vie" ? → NON
- Peut-on s'arrêter après "Mais dans ses yeux..." ? → NON
- Le CTA apparaît-il comme une évidence ? → OUI

✅ **Résultat** : Flow narratif maintenu jusqu'à l'action
