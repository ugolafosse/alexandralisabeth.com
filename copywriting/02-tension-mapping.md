# Tension Mapping - Analyse des tensions de l'audience

**Tactique** : Explorer les conflits de l'audience pour créer un copywriting qui résout leurs tensions

---

## 1. Définition de l'audience

**Ils sont** : Lecteurs de fantasy épique francophones (25-45 ans)
**Ils ont besoin** : D'histoires immersives et originales qui les transportent
**Pour que** : Ils puissent s'évader du quotidien et vivre des émotions intenses

---

## 2. Tensions qui les éloignent du produit

| Conflit/Tension | Solution du livre |
|-----------------|-------------------|
| **"Encore une fantasy générique avec des dragons et des châteaux"** | ✅ Univers **primordial chamanique**, pas de clichés médiévaux |
| **"Je n'ai pas de temps, les livres sont trop longs"** | ✅ "Réservez votre week-end" = page-turner addictif |
| **"Les nouveaux auteurs sont risqués"** | ✅ **Social proof** : 10 reviews 4.8/5 |
| **"La fantasy c'est compliqué, trop de noms bizarres"** | ✅ Personnages attachants, noms simples (Tukum, Maek) |
| **"Je ne veux pas m'attacher si la suite n'existe pas"** | ⚠️ **Tension réelle** : Suite existe mais attente frustrante |
| **"Les ebooks c'est cher pour un auteur inconnu"** | ✅ **5,99€** + prologue gratuit + Kindle Unlimited |
| **"Je ne sais pas si ça va me plaire"** | ✅ **Prologue gratuit** sur le site |
| **"Fantasy c'est pour les ados"** | ✅ Thèmes matures : dilemme moral, spiritualité |

## 3. Tensions émotionnelles (profondes)

| Tension intrinsèque | Comment le livre y répond |
|---------------------|---------------------------|
| **Besoin de sens/connexion** | Thème des "âmes connectées à quelque chose de plus grand" |
| **Nostalgie du primitif/naturel** | "Retour aux sources", "bouffée d'oxygène" |
| **Désir d'évasion totale** | "Fiction devient réalité tangible", "comme un film" |
| **Besoin de héros complexes** | Maek = Âme Noire qui lutte contre son destin |

## 4. Applications dans le copywriting

### Tension : "Fantasy générique"
**Solution copywriting** :
```
De la forêt à perte de vue, des clans vivant en harmonie avec la
nature sauvage... et pourtant, un équilibre millénaire sur le point
de se briser.
```
→ Différenciation claire sans dire "pas comme les autres"

### Tension : "Auteur inconnu = risque"
**Solution copywriting** :
- Testimonials section avec vraies reviews Amazon
- Lien direct vers tous les avis Amazon
- Prologue gratuit pour essai sans risque

### Tension : "Pas le temps"
**Solution copywriting** :
Citation review : "Un conseil, réservez votre week-end !"
→ Implication que c'est une lecture binge-able

### Tension : "Fantasy = immature"
**Solution copywriting** :
```
Le loup vous fixe. Vous devez prendre sa vie pour survivre.
Mais dans ses yeux, vous voyez... un ami.
```
→ Dilemme moral mature, pas de "épée magique" enfantine

---

## 5. Insights clés

✅ **Ne pas nier** ("Pas de dragons") → **Affirmer** ("La forêt, les clans, les esprits")
✅ **Montrer la complexité** plutôt que dire "c'est profond"
✅ **Social proof via citations**, pas via chiffres faibles
✅ **Prologue gratuit** = meilleure réponse au risque perçu
